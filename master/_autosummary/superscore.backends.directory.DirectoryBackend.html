superscore.backends.directory.DirectoryBackend

class superscore.backends.directory.DirectoryBackend(path, cfg_path=None)[source]

Bases: _Backend

Backend that stores each Entry as an individual file. Files are distributed into directories according to the UUID of their Entry, to keep the number of files in each directory reasonably small.

search(*search_terms)[source]

Yield Entry objects matching all search_terms. Each SearchTerm has the format (<attr>, <operator>, <value>). Some operators take tuples as values.

The supported operators are: - eq (equals) - lt (less than or equal to) - gt (greater than or equal to) - in - like (fuzzy match, depends on type of value)
