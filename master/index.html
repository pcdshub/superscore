Introduction

Superscore is an EPICS PV configuration management application that aims to be searchable and flexible. Superscore features a centralized data model, multiple communication layers (CA, PVA), a python client, and a Graphical User Interface. The goal is to provide users a the ability to snapshot the state of their system and restore known good states. Central to the design of this application is enabling users to specify what groups of PVs they want to store, and organize them in a sensible way.
